# shopAPI

## Technologies:


- [x] Node.js
- [x] Express.js
- [x] TypeScript
- [x] MongoDB

## Usage:

- [x] POST api/v1/login - login user
- [x] POST api/v1/register - register user
- [x] GET api/v1/products - get all products from DB
- [x] POST api/v1/product - create new product 
- [x] GET api/v1/logout - logout user
